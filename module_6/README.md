Цель проекта - собрать, проанализировать и обработать данные с сайта авто.ру. На основе этих данных создать модель, предсказывающую стоимость автомобиля.<br>
В задании нам был предоставлен только тестовый набор данных, в котором не было колонки цена. И предлагалось самим собрать данные для обучения модели. Проанализировав тестовый набор данных, мы определили какие данные нам надо собрать и собрали их с сайта auto.ru (потому что тестовые данные собраны с этого сайта)<br>
